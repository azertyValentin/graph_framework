L'implementation de l'algotithme de Prim se situe dans le fichier GraphToolsList (dans le dossier GraphAlgorithms).
Afin de lancer l'algorithme, vous devez lui fournir un graph ainsi que le noeud de départ.
L'algorithme a pour sortie une pair contenant le cout et les neouds dans l'ordre de parcours.
La complexité est en O(n*h) avec n le nombre d'arrête et h la hauteur de l'abre.
