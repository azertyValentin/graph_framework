L'implementation du binaryHeap se situe dans le fichier du même nom (dans le dossier GraphAlgorithms).

Nous avons implémenté une version du BinaryHeap. La complexité de l'algorithme est la suivante:
Space		O(n)        (chaque noeud est stocké)
Search		O(n)        (dans le pire des cas, on parcours l'ensemble de l'abre)
Insert		O(log n)
Find-min	O(1)
Delete-min	O(log n)

Dijkstra

L'implementation de l'algorithme de Dijkstra se situe dans le fichier GraphToolsList (dans le dossier GraphAlgorithms).
Afin de lancer l'algorithme, vous devez lui fournir un graph ainsi que le noeud de départ.
L'algorithme a pour sortie une pair contenant le liste des couts et des prédecesseurs.
